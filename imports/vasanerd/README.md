# VasaNerd-export

Lägg en tillåten JSON- eller JavaScript-export här via GitHubs webbsida. Kör sedan
**Importera historik från VasaNerd** med metoden `uploaded_export`. Adaptern hittar
resultattabeller rekursivt och sparar okända originalfält i SQLite.

Publicera inte en full kopia utan att källans villkor och attribution är klarlagda.
